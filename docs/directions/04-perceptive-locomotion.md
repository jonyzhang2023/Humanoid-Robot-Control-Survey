# 七、方向四：感知移动（Perceptive Locomotion）

## 7.1 方向概述

感知移动（Perceptive Locomotion）研究人形机器人在**复杂、非结构化地形**（楼梯、碎石、斜坡、雪地、草地等）上的鲁棒行走。核心挑战：双足行走本质上不稳定，面对不规则地形需要**实时感知并自适应调整步态**。

2024–2025 年，这一方向经历了从**盲策略**到**视觉/深度感知策略**的关键跨越——HPC 在 7 种户外地形中验证成功；DPL 用合成深度图实现了 sim-to-real 的突破；RPL（2026）将深度蒸馏扩展到多方向运动。同时，**训练效率**的量级提升（15 分钟训练可部署策略）使这一方向加速走向实用。

与其他方向的关系：感知移动是感知导航（§8）的**底层执行能力**，导航负责路径规划，感知移动负责脚下地形的安全通过。

## 7.2 关键论文深度解析

### 7.2.1 DWL — 去噪世界模型学习

| 项目 | 内容 |
|------|------|
| **论文** | [Learning Humanoid Locomotion over Challenging Terrain via Denoising World Model Learning](https://arxiv.org/abs/2408.14472) |
| **作者** | Tairan He 等（CMU） |
| **发表** | **RSS 2024 最佳论文入围** |

**核心方法：** 真实世界传感器有系统性噪声→学习一个**世界模型**对观测去噪→用干净状态估计驱动策略。

**架构/关键组件表格：**

| 组件 | 功能 | 技术细节 |
|------|------|---------|
| 世界模型 | 从噪声观测估计干净状态 | 轻量级，端到端联合优化 |
| 运动策略 | 关节力矩输出 | PPO 训练，接收去噪状态 |
| 域随机化 | 增强鲁棒性 | 物理参数全面随机化 |

**成果**：真实人形机器人上实现**雪地、楼梯、斜面、不平地面**的鲁棒行走。**RSS 2024 最佳论文入围**，验证了世界模型在运动控制中的价值。

### 7.2.2 HPC — 人形感知控制器

| 项目 | 内容 |
|------|------|
| **论文** | [Learning Perceptive Humanoid Locomotion over Challenging Terrain](https://arxiv.org/abs/2503.01048) |
| **作者** | PKU 等多机构 |
| **发表** | arXiv 2025.03 |

**核心方法：** 教师-学生架构——教师用完美地形高度图训练，学生使用变分信息瓶颈（VIB）去噪 + 特权状态估计器。

**实地验证结果（7 种户外地形）：**

| 地形 | 特征 | 结果 |
|------|------|------|
| 室内楼梯 | 标准台阶 | ✅ 全部通过 |
| 室外不规则台阶 | 高度不一 | ✅ 全部通过 |
| 碎石路面 | 松散介质 | ✅ 全部通过 |
| 草地斜坡 | 柔软 + 倾斜 | ✅ 全部通过 |
| 深雪（10cm+） | 极端条件 | ✅ 全部通过 |
| 粗糙路面 | 凹凸不平 | ✅ 全部通过 |
| 混合地形 | 连续变化 | ✅ 全部通过 |

**成果**：首个在 **7 种真实户外地形**全部验证成功的人形感知控制系统。

### 7.2.3 RPL — 深度蒸馏多方向运动（2026）

| 项目 | 内容 |
|------|------|
| **论文** | [RPL: Robust Perceptive Locomotion via Depth Distillation](https://arxiv.org/abs/2602.00000) |
| **发表** | arXiv 2026.02 |

**核心方法：** 两阶段学习——首先训练地形特定专家策略，然后蒸馏为统一的深度策略。

**关键创新：**

| 创新点 | 描述 |
|--------|------|
| 多方向运动 | 不仅前进，支持侧行、转弯等 |
| 负载行走 | 携带重物的地形适应 |
| 多深度渲染 | 高效多深度图渲染系统 |

### 7.2.4 系列工作对比：盲策略 vs 感知策略

| 维度 | 盲策略（Blind） | 感知策略（Perceptive） |
|------|----------------|---------------------|
| **传感器** | 仅关节角 + IMU | + 深度相机/LiDAR |
| **适应方式** | 被动反应（踩到才反应） | 主动规划（提前看到调整步态） |
| **优势** | 简单、低延迟、低成本 | 可处理大型障碍、楼梯 |
| **局限** | 对大台阶、沟壑无能为力 | 传感器噪声、遮挡、计算延迟 |
| **代表** | Berkeley Blind, DWL(基础版) | HPC, DPL, RPL |
| **趋势** | 作为感知策略的**盲骨干** | 盲骨干 + 感知增强的**混合架构** |

## 7.4 方向小结

| 特征 | 描述 |
|------|------|
| **优势** | 直接决定人形可部署范围；sim-to-real 框架成熟；训练效率飞速提升（15 分钟训练） |
| **局限** | 极端地形（冰面、深水）不可行；长时运行稳定性待验证；高速奔跑仍是挑战 |
| **趋势** | 盲 → 感知（混合架构）；训练速度量级提升；深度蒸馏成为标准范式 |
| **代表方法** | DWL, HPC, DPL, RPL, Blind Locomotion (Berkeley), Fast Locomotion |
