# 人形机器人运动控制完整研究报告

> 覆盖 2016–2026 年，~150 篇论文，5 大研究方向的系统性深度调研

## 内容概览

本项目是对**人形机器人运动控制（Humanoid Robot Motion Control）**领域的系统性深度调研，覆盖从 DeepMimic（2018）到 2026 年 2 月的最新进展，重点聚焦运动控制的五大前沿方向。

### 报告结构

| 部分 | 内容 |
|------|------|
| **第一部分：认知建立** | 执行摘要、核心定义与数学框架、发展历程时间线 |
| **第二部分：五大研究方向** | ① 通用全身控制 ② 人-场景交互 (HSI) ③ 人-物交互 (HOI) ④ 感知移动 ⑤ 感知导航 |
| **第三部分：生态与产业** | 全球企业与机构布局、开源生态全景 |
| **第四部分：分析与展望** | 技术路线对比、评估基准、趋势与未来 |
| **附录** | 论文与参考文献总索引（~150 篇） |

### 核心数字

- 📄 **~150 篇** 代表性论文（含 arXiv 链接）
- 🔬 **5 个** 研究方向深度解析
- 🏢 **20+ 家** 全球企业与机构布局分析
- 🤖 **12+** 硬件平台对比
- 📦 **10+** 核心开源项目

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```

```
├── docs/                          # VitePress 网站内容
```

## 更新机制

- 网站多文件为**主版本**（source of truth）
- 每月更新对应方向的内容页面
- Push 到 `main` 分支后 GitHub Actions 自动构建并部署
