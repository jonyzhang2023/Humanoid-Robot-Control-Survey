# 方向四：感知移动

### 7.1 方向概述

感知移动（Perceptive Locomotion）研究人形机器人如何在**复杂、非结构化地形**（楼梯、碎石、斜坡、雪地、草地等）上实现鲁棒行走。其核心挑战在于：双足行走本质上是不稳定的动态系统，面对不规则地形时，机器人需要**实时感知地形并自适应调整步态**。

2024–2025 年，这一方向经历了从**盲策略（Blind Policy）**到**视觉/深度感知策略（Perceptive Policy）**的关键跨越——机器人不再仅依靠本体感受（关节角度、IMU），而是开始利用深度相机或 LiDAR 主动感知地形。

> 📖 综述 [Sim-to-Real in Locomotion: A Survey](https://arxiv.org/abs/2507.04542) 系统分析了腿式机器人运动控制的 sim-to-real 迁移技术。

### 7.2 关键论文深度解析

#### 7.2.1 DWL — 去噪世界模型学习

| 项目 | 内容 |
|------|------|
| **论文** | [*DWL: Denoising World Model Learning for Humanoid Locomotion*](https://arxiv.org/abs/2408.14472) |
| **发表** | RSS 2024 最佳论文入围 |
| **机构** | CMU |
| **日期** | 2023 |

**核心洞见**：真实世界中传感器（IMU、关节编码器）存在系统性噪声，直接使用带噪声观测训练的策略在真机上性能严重下降。DWL 提出**通过学习一个世界模型来去噪**。

**方法**：
- 训练一个轻量世界模型，从带噪声的观测历史中估计干净的特权状态
- 将去噪后的状态估计作为策略的输入
- 世界模型和策略联合端到端优化

**成果**：
- 在真实人形机器人上实现了鲁棒行走，对外部推击、地形变化具有强适应性
- RSS 2024 最佳论文入围，验证了世界模型在运动控制中的价值

#### 7.2.2 HPC — 人形感知控制器

| 项目 | 内容 |
|------|------|
| **论文** | [*Learning Perceptive Humanoid Locomotion over Challenging Terrain*](https://arxiv.org/abs/2503.01048) |
| **机构** | PKU / 多机构 |
| **日期** | 2025.03 |

**核心方法**：集成以机器人为中心的地形感知与地形去噪的人形感知控制器（HPC）。

**系统架构**：
1. **教师策略**（无噪声训练）：使用完美地形高度图 + 特权状态训练，产生 oracle 动作
2. **学生策略**（带噪声训练）：
   - **变分信息瓶颈（VIB）** 对传感器数据去噪
   - **特权状态估计器** 从历史观测估计不可直接获取的状态
   - 学习模仿教师的 oracle 动作
3. **地形感知模块**：使用深度相机获取机器人前方地形的高度图

**实际验证（7 种户外地形）**：

| 地形类型 | 成功率 | 特征 |
|---------|--------|------|
| 室内楼梯 | ✅ | 标准台阶 |
| 室外台阶（不规则） | ✅ | 高度不一 |
| 碎石路面 | ✅ | 松散介质 |
| 草地斜坡 | ✅ | 柔软+倾斜 |
| 深雪（10cm+） | ✅ | 极端条件 |
| 粗糙路面 | ✅ | 凹凸不平 |
| 混合地形 | ✅ | 连续变化 |

#### 7.2.3 DPL — 纯深度感知人形运动

| 项目 | 内容 |
|------|------|
| **论文** | [*DPL: Depth-Only Perceptive Humanoid Locomotion*](https://arxiv.org/abs/2510.13809) |
| **日期** | 2025.10 |

**核心方法**：
- **合成深度训练**：在仿真中合成逼真的深度图像（而非理想高度图），直接用于策略训练
- **交叉注意力地形重建**：使用多模态交叉注意力 Transformer 从深度图重建地形表示
- **盲骨干+地形感知融合**：将纯本体感受的盲策略与地形感知模块融合

**关键发现**：
- 使用合成深度图（而非理想高度图）训练显著提升 sim-to-real 迁移效果
- 交叉注意力机制有效融合本体感受和外感受信息

### 7.3 盲策略 vs 感知策略

| 维度 | 盲策略（Blind） | 感知策略（Perceptive） |
|------|----------------|---------------------|
| **传感器** | 仅本体感受（关节角+IMU） | + 深度相机/LiDAR |
| **适应方式** | 被动反应（踩到障碍才反应） | 主动规划（提前看到地形调整步态） |
| **优势** | 传感器简单、延迟低、价格低 | 可处理大型障碍、楼梯等 |
| **局限** | 对大台阶、沟壑无法应对 | 传感器噪声、遮挡、计算延迟 |
| **代表** | Causal Transformer (Berkeley), Lee et al. | HPC, DPL |
| **适用场景** | 轻度不平整地面 | 复杂户外地形、楼梯 |

> 🔍 **趋势**：盲策略和感知策略并非对立——最新工作（如 HPC、DPL）采用**盲骨干+感知增强**的混合架构，在基本行走能力上叠加地形感知。

### 7.4 Sim-to-Real 关键技术

| 技术 | 作用 | 代表工作 |
|------|------|---------|
| **域随机化** | 随机化物理参数，增强策略鲁棒性 | 几乎所有工作 |
| **教师-学生蒸馏** | 教师用特权信息训练，学生仅用真实可获取信息 | HPC, DWL |
| **世界模型去噪** | 学习世界模型估计干净状态 | DWL, HPC |
| **课程学习** | 逐步增加地形难度 | 大多数地形训练 |
| **系统辨识** | 自动调整仿真参数匹配真实机器人 | ASAP |
| **残差适配** | 在真机上学习残差修正 | MOSAIC |

### 7.6 方向小结

| 特征 | 描述 |
|------|------|
| **优势** | 直接决定人形机器人的可部署范围；sim-to-real 框架成熟；开源代码和平台丰富 |
| **局限** | 极端地形（冰面、深水）仍不可行；长时运行稳定性待验证；高速奔跑仍是挑战 |
| **趋势** | 从盲策略 → 感知策略；地形去噪技术成为标配；训练速度急剧提升（15 分钟训练） |
| **代表方法** | DWL, HPC, DPL, Blind Locomotion (Berkeley), Fast Humanoid Locomotion |
